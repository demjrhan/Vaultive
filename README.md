## Build Instructions

### Navigate to the backend directory  
- cd Vaultive/Project/Backend
- mkdir Migrations
- dotnet restore  
- dotnet build  
- dotnet ef migrations add InitialCreate  
- dotnet ef database update  
- dotnet run
